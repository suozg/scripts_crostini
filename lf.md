# 📂 ДОВІДКА ПО ГАРЯЧИХ КЛАВІШАХ LF
---

## 🔎 НАВІГАЦІЯ ТА ПОШУК

| Клавіші | Дія |
| :--- | :--- |
| `/` | **FZF Search**: швидкий пошук та перехід |
| `gh` | Перейти в папку `~/WORK` |
| `gs` | Перейти на **Google Drive** |
| `Z` | Призупинити lf (повернення в термінал, назад — `fg`) |
| `m` | Встановити мітку (mark) |
| `'` | Вибрати мітку |
| `"` | Видалити мітку |
| `4j` | Стрибок на 4 рядки вниз |
| `45gg` | Перейти відразу до 45-го рядка |

---

## 📝 РОБОТА З ФАЙЛАМИ

| Команда | Опис |
| :--- | :--- |
| `l` / `Right` | Відкрити файл або увійти в каталог |
| `a` | Перейменувати (курсор у кінці назви) |
| `A` | **Масове перейменування** (редагування списку у Vim) |
| `n` | Створити новий каталог (`mkdir`) |
| `N` | Створити новий порожній файл (`touch`) |
| `Cs` | Виділити файли за маскою |
| `Ca` | Виділити всі файли в директорії |

---

## ✂️ КОПІЮВАННЯ ТА ВИДАЛЕННЯ

| Команда | Дія |
| :--- | :--- |
| `dd` | Вирізати (cut) |
| `yy` | Копіювати (yank) |
| `pp` | Вставити (paste) |
| `db` | Перемістити в **КОШИК** |
| `dr` | Переглянути кошик та відновити файл |
| `de` | Очистити кошик повністю |
| `dD` | **ЗНИЩИТИ НАЗАВЖДИ** |

---

## ⚙️ АВТОМАТИЗАЦІЯ (CUSTOM)

| Команда | Опис |
| :--- | :--- |
| `Cd` | Конвертувати `.odt` в `.docx` |
| `Cp` | Стиснути PDF (`shrink_pdf.sh`) |

---

## 🛠️ СТАНДАРТНІ КЛАВІШІ (DEFAULT)

| Рух | Дія |
| :--- | :--- |
| `j` / `k` | Вниз / Вгору |
| `h` | На рівень вище (батьківська папка) |
| `gg` / `G` | У початок / У кінець списку |
| `Ctrl+u / d` | Прокрутка на півсторінки |

| Інше | Дія |
| :--- | :--- |
| `Space` | Виділити файл (Mark) |
| `v` | Інвертувати виділення |
| `zh` | Показати/приховати приховані файли |
| `q` | Вихід з програми |
| `:` | Ввести команду вручну |
| `ESC` | Оновити вміст / Скасувати вибір |
