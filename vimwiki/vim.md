## 1. СПЕЦІАЛЬНІ ФУНКЦІЇ ТА ЕКСПОРТ

| Клавіші | Опис |
| :--- | :--- |
| `\b` | Швидкий експорт у PDF (standard.html) |
| `\bt` | Експорт у PDF з вибором шаблону зі списку |
| `\rn` | (LSP) Перейменувати змінну/функцію всюди |
| `F6` | Увімкнути/вимкнути видимість спецсимволів |
| `F7` | Увімкнути/вимкнути перенос рядків (wrap) |
| `F8` | Перемикання нумерації рядків (relative/absolute) |
| `F9` | Увімкнути/вимкнути перевірку орфографії |

## 2. РЕЖИМИ ТА НАВІГАЦІЯ

| Команда | Опис |
| :--- | :--- |
| `i` | Режим вставки (Esc — вихід) |
| `v` / `V` / `^v` | Виділення: символи / рядки / блоки |
| `h` `j` `k` `l` | Рух: вліво, вниз, вгору, вправо |
| `w` `e` `b` | До слова вперед / кінець слова / назад |
| `0` `^` `$` | Початок рядка / перший символ / кінець |
| `H` `M` `L` | Верх / середина / низ екрана |
| `gg` / `G` | Початок файлу / кінець файлу |
| `:n` | Перейти до рядка номер **n** |
| `Ctrl+d / u` | Прокрутка вниз / вгору на пів екрана |
| `Ctrl+f / b` | На екран уперед / назад |
| `/текст` | Пошук уперед (`?` — назад) |
| `n` / `N` | Наступний / попередній збіг |
| `\1` | Відкрити/Закрити поточний файл (Toggle) |
| `Ctrl+g` | Статус файлу та позиція курсора |

## 3. ПОШУК ТА ФАЙЛОВИЙ МЕНЕДЖЕР (FZF / Netrw)

| Команда | Опис |
| :--- | :--- |
| `\ff` | Пошук файлів у проекті (FZF) |
| `\fg` | Пошук тексту (Live Grep / Ripgrep) |
| `\fb` | Пошук по списку відкритих буферів |
| `\fh` | Історія нещодавно відкритих файлів |
| `\z` | Виправити слово (Spell Suggestion) |
| `\e` | Відкрити/закрити провідник файлів (Lexplore) |

**У вікні FZF:**
* `Enter` / `Ctrl+t` — Відкрити у новій вкладці.
* `Ctrl+x` / `v` — Горизонтальний / Вертикальний спліт.

## 4. БУФЕРИ ТА ВІКНА (SPLITS)

| Буфери | Дія |
| :--- | :--- |
| `:e файл.txt` | Відкрити файл у новий буфер |
| `:ls` | Список усіх буферів |
| `:bn` / `:bp` | Наступний / Попередній буфер |
| `Ctrl+j / k` | Швидке перемикання буферів |
| `:bd` | Закрити поточний буфер |

| Split (Вікна) | Дія |
| :--- | :--- |
| `:sp` / `:vsp` | Горизонтальний / Вертикальний спліт |
| `Ctrl+w w` | Перемикання між вікнами |
| `Ctrl+w h/j/k/l` | Рух між вікнами (ліво/низ/верх/право) |
| `Ctrl+w q` | Закрити вікно |
| `Ctrl+w =` | Вирівняти розміри вікон |

| `gc` / `gcc` | Закоментувати виділене / рядок |
