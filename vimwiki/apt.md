# Apt — менеджер пакетів для Debian

## Основні команди

| №  | Дія                                             | Команда                                |
|----|-------------------------------------------------|----------------------------------------|
| 1  | Оновлення списку пакетів                        | `sudo apt update`                      |
| 2  | Оновлення всіх встановлених пакетів             | `sudo apt upgrade`                     |
| 3  | Встановлення пакета                             | `sudo apt install <package>`           |
| 4  | Видалення пакета (залишає конфігураційні файли) | `sudo apt remove <package>`            |
| 5  | Повне видалення пакета разом з конфігурацією    | `sudo apt purge <package>`             |
| 6  | Перевірка інформації про пакет                  | `apt show <package>`                   |
| 7  | Пошук пакета                                    | `apt search <package>`                 |
| 8  | Перевірка, які пакети ще потрібні               | `apt depends <package>`                |
| 9  | Перевірка, яким додаткам цей пакет потрібен     | `apt rdepends <package>`               |
| 10 | Очищення кешу завантажених пакетів              | `sudo apt clean`                       |
|    |                                                 | `sudo apt autoclean`                   |
| 11 | Автоматичне видалення непотрібних пакетів       | `sudo apt autoremove`                  |

---

## Трюки і корисні фішки

| № | Дія                                         | Команда                                              |
|---|---------------------------------------------|------------------------------------------------------|
| 1 | Видалення всіх залишкових конфігів (rc)     | `sudo apt purge $(dpkg -l \| awk '/^rc/{print $2}')` |
| 2 | Додаткові прапори                           | `-y`           # автоматично yes                     |
|   |                                             | `--simulate`   # перевірка                           |
| 3 | Підрахунок розміру без встановлення пакету  | `apt show gedit \| grep Installed-Size`              |
| 4 | Використання apt для перевірки історії змін | `grep "install" /var/log/dpkg.log`                   |
| 5 | Перевірка, чи пакет встановлений            | `dpkg -l \| grep <package>`                          |
| 6 | Видалення пакета без очищення залежностей   | `sudo dpkg --remove <package>`                       |
| 7 | Видалення пакета з очищенням конфігурацій   | `sudo apt purge <package> && sudo apt autoremove`    |


---

## Поради

* Використовуй `purge` для повного очищення системи від пакета.
* Регулярно перевіряй `rc` пакети, щоб не накопичувалися залишки.
* Для великих змін завжди роби `apt update` перед `upgrade`.
* Для тестування нових пакетів використовуй `--simulate`.
* Домашні файли користувача (`~/.config`, `~/.vimrc`) apt не видаляє.
