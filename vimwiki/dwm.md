# DWM Шорткати

---

## Основні комбінації для керування вікнами

| Комбінація клавіш  | Дія                                        |
|--------------------|--------------------------------------------|
| Mod + b            | Показати/сховати панель                    |
| Mod + j            | Фокус на наступне вікно                    |
| Mod + k            | Фокус на попереднє вікно                   |
| Mod + i            | Збільшити кількість головних вікон         |
| Mod + d            | Зменшити кількість головних вікон          |
| Mod + h            | Зменшити ширину головного вікна            |
| Mod + l            | Збільшити ширину головного вікна           |
| Mod + Return       | Збільшити/повернути фокус на головне вікно |
| Mod + Tab          | Перемкнути на попередню вкладку (тег)      |
| Mod + Shift + c    | Закрити вікно                              |
| Mod + t            | Макет "головне + стек"                     |
| Mod + m            | Повноекранний макет                        |
| Mod + f            | Плаваючий макет для наступного вікна       |
| Mod + Space        | Перемикання між останніми двома макетами   |
| Mod + 0            | Показати всі теги                          |
| Mod + Shift + 0    | Призначити всі теги поточному вікну        |
| Mod + 1..9         | Перемикання на тег 1..9                    |
| Mod + Shift + 1..9 | Призначити тег 1..9 поточному вікну        |

---

## Комбінації для запуску і системних дій

| Комбінація клавіш   | Дія                                     |
|---------------------|-----------------------------------------|
| Mod + p             | Відкрити DMenu (spawndmenu)             |
| Mod + Ctrl + t      | Змінити тему (світла/темна)             |
| Mod + Shift + Enter | Запуск терміналу                        |
| Ctrl + Space        | Перемкнути розкладку (toggle_layout.sh) |
| Mod + x             | Відкрити clipmenu                       |
| Mod + Shift + x     | Очистити буфер (clipdel -d .)           |
| Mod + z             | Отримати вміст буфера (get_clipb.sh)    |
| Mod + Shift + z     | Відправити вміст буфера (send_clipb.sh) |
| Mod + /             | Відкрити довідку (~/awards/*_help.txt)  |
| Mod + \             | Відкрити завдання (~/awards/events.txt) |
| Mod + r             | Відкрити інтернет-радіо                 |
| Mod + n             | Запустити awardstart                    |
| Mod + Shift + n     | Запустити nn_start                      |
| Mod + Shift + q     | Вийти з DWM                             |

---

## Комбінації для роботи з моніторами

| Комбінація клавіш | Дія                                     |
|-------------------|-----------------------------------------|
| Mod + ,           | Фокус на попередній монітор             |
| Mod + .           | Фокус на наступний монітор              |
| Mod + Shift + ,   | Перемістити вікно на попередній монітор |
| Mod + Shift + .   | Перемістити вікно на наступний монітор  |
